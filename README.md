# 🧠 Quiz Application

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript. This app allows users to answer multiple-choice questions with a timer and track their progress.

---

## 🚀 Features

* 📌 Multiple-choice quiz questions
* ⏱️ Timer for each question
* 📊 Question counter (e.g., 1/10)
* ✅ Instant answer selection
* ➡️ Next button navigation
* 🎯 Score tracking (can be added if not yet implemented)
* 💻 Responsive and clean UI

## ⚙️ How It Works

1. Quiz starts from Question 1
2. Timer begins for each question
3. User selects an answer
4. Click **Next** to move forward
5. At the end, score can be displayed
